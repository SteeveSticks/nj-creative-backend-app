# NJ Creative Firm Backend

A Node.js + Express + TypeScript backend for NJ Creative Firm.
Features: Contact API, Blog API (CRUD), Cloudinary uploads, Admin JWT auth, and email notifications.

## Setup

- Copy `.env.example` to `.env` and fill values.
- Install dependencies: `npm install`
- Run in dev: `npm run dev`
- Build: `npm run build`
- Start: `npm start`
